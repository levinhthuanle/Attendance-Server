- Login Screen
	- Đã đủ cơ bản, chưa có mở rộng

### Student Graph
- Home Screen
	- Lấy thông tin chung của sinh viên - /student/current
	- Lấy attendance record của sinh viên (tất cả) - /student/all/attendance_records
	- Kiểm tra sinh viên enroll trong class hay ko - /student/check-enrollment/{class_id}

- Confirm Screen
	- Lấy thông tin chung của một lớp sinh viên tham gia - /class/{class_id}/information
	- Nhận session id từ client (Server sẽ đánh dấu điểm danh sinh viên session này) - /student//roll_call

- Classes Screen
	- Lấy danh sách lớp sinh viên đang enroll - /student/all/class/enroll

- Classes Detail Screen
	* Lấy thông tin của một lớp sinh viên tham gia OK  - /class//{class_id}/information 
	- Lấy attendance record của sinh viên (theo lớp) /student/{class_id}/all/attendance_records

- Profile Screen
	* Lấy thông tin chung của sinh viên - /student/current

### Teacher Graph
- Home Screen
	- Lấy thông tin chung của giáo viên - /teacher/current
	- Lấy session giáo viên đã dạy OK - /teacher/teaching_sessions/

- Classes Screen
	- Lấy class giáo viên đang dạy - /teacher/teaching_classes

- Class Detail Screen
	- Lấy thông tin chung của một lớp giáo viên dạy - /class/{class_id}/information
	- Lấy danh sách sinh viên tham gia class - /class/{class_id}/students
	- Lấy danh sách session của class - /class/{class_id}/sessions
	- Tạo và trả về session id mới - /session/

- Take Attendance Screen
	- Lấy danh sách sinh viên attend session - /record/{session_id}
	- Lấy danh sách sinh viên absent from session (cái này với cái trên có thể là 1) - /record/{session_id}

- Class Students Screen
	* Lấy danh sách sinh viên tham gia class
	* Lấy record của sinh viên theo class
	* Lấy danh sách session của class

- Class Sessions Screen
	* Lấy danh sách sinh viên attend session
	* Lấy danh sách sinh viên absent from session (cái này với cái trên có thể là 1)

- Profile Screen
	* Lấy thông tin chung của giáo viên
